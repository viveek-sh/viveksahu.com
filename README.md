## 🛠 Running Locally

```bash
# 1. Clone the repo
git clone https://github.com/viveek-sh/viveksahu.com.git
cd viveksahu.com

# 2. Install dependencies
npm install

# 3. Start dev server
npm run dev
```

Open [`http://localhost:4321`](http://localhost:4321) in your browser.

> Astro runs on port **4321** by default.
